A handle to an asset and the handle to where it will be moved.

## Parameters {#section-3cf94fa46d3848729dc841b123a1cea7}

|  Name  | Type  | Description  |
|---|---|---|

|  `*`folderHandle`*`  | `xsd:string`  | The handle to the destination folder.  |

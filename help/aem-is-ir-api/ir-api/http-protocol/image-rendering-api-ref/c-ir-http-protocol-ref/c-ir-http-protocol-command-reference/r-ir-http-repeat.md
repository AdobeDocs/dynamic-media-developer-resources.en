## Properties {#section-262bf540930d4890b678ea00cefe1909}

Material attribute. Ignored by solid color, decal, and cabinet materials.

## Default {#section-e5bbd7d9fbb74852849e605d20f550bb}

`catalog::Repeat`, if the material is based on a catalog entry, otherwise `0` (straight repeat).
